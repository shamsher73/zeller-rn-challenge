This is an react native project created with expo.

## Get started

1. Install dependencies

   ```bash
   npm install
   ```

2. Start the app

   ```bash
    npm run start
   ```

   OR

   ```bash
    npm run ios
   ```

   OR

   ```bash
    npm run android
   ```

3. Run test cases

   ```bash
    npm run test
   ```

   Tests are in

   ```bash
   /components/__tests__
   /components/home/__tests__
   /utils/__tests__
   ```
